D&D 4e File Viewer

This app allows you to view D&D 4e character builder files on your iOS device. After opening the file, you can browse your skills, feats, powers, racial and class features. When you open the app for the first time, I've populated it with some sample characters. You can view these by tapping on them.

Throughout the app, you might see hyperlinks like this one that will take you to a detailed explanation of that link. For example, open up 'kaeltor12', go to 'Ability Scores' and click on 'Dexterity'. There you'll see a breakdown of how that ability score was determined. Where possible, the app tries to display the full information regarding the Feat, Trait, Feature, Skill, or Power - instead of an abbreviated blurb. Because 'page space' is unlimited in the digital realm, we can simplify what would otherwise be a 20-page character sheet into a very condensed space.

How-To Import

This app registers the '.dnd4e' extension as being viewable on your phone. In any app that allows you to view files or attachments you can open up the character builder files. Two examples are shown below. Whenever you see the same Icon, you should be able to open it up.

How-To Delete

Done with a character? Import the same character with a new level? Get rid of the old one by sweeping left-to-right on the file name.

How-To Navigate

Drilled really deep, and need to get back to the main Character menu? Never fear, just swipe right-to-left on the title bar at the top, and you'll be taken back to the character main page.

How-To Use the Compendium

As mentioned earlier, certain pages have links to descriptions. Some links might take you to the DnD Insider Compendium page. If you don' have a DnD Insider account, don't worry. The information contained in the file should be enough for you go on.
